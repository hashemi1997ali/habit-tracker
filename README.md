# 🧠 Habit Tracker

A modern habit tracking app built with React (Vite), designed to help users stay consistent and track their daily progress.

---

## ✨ Features

- ➕ Add new habits with custom goals
- 🗑️ Delete habits
- 🔄 Reset habit progress
- 📈 Track daily completion status
- 🎯 Dynamic progress calculation
- 💾 Persistent data using Local Storage
- 🎨 Clean and responsive UI with Tailwind CSS + DaisyUI
- 🎯 Font Awesome icons for better UX

---

## 🧩 Components Overview

- **Header** → App title
- **HabitStats** → Overall progress summary
- **HabitForm** → Add new habits
- **HabitList** → Displays habits
- **HabitListItem** → Individual habit card
- **ProgressAction** → Increment / decrement controls

---

## ⚙️ Tech Stack

- React (Vite)
- Tailwind CSS
- DaisyUI
- Font Awesome
- Local Storage
